## Agent skills

### Issue tracker

Issues live in GitHub Issues (`https://github.com/aoaoAi2026/math_samll`), using the `gh` CLI. See `docs/agents/issue-tracker.md`.

Uses the default canonical labels: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repo — one `CONTEXT.md` + `docs/adr/` at root. See `docs/agents/domain.md`.
